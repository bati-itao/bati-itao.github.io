Introduction: Zoom and responsive design

It is important to allow web content to scale effectively so that the layout can be optimized at different viewport widths. People with low vision need to enlarge text and read it in a single column without horizontal scrolling. This reflow of content is important to enable reading and tracking from one line to the next.

Everyone benefits from responsive design including:

  • People with low vision can increase text size in content so that they can read it.
  • People who use mobile browsers or zoom can read content easily without scrolling sideways.
  • People who use fixed screens and fixed orientation.
