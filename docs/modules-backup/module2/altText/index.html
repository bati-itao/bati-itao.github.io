Alt Text: Alternative Text for Images

Alternative Text for Images in Documents

What is alt text?

Alt text (alternative text) is a short, descriptive alternative presented to users who cannot see an image. It conveys the image’s purpose and content, ensuring everyone can access the information, regardless of ability.

Why alt text matters in Microsoft 365

Alt text is essential for making documents accessible in Microsoft 365 applications such as Word, PowerPoint, Excel, and Outlook. It ensures that users of assistive technologies can understand the content and purpose of images, charts, and graphics in non-web documents. Well-written alt text improves discoverability, usability, and compliance with accessibility standards.

How to add alt text in Microsoft 365

  • Word & PowerPoint: Right-click the image, shape, or chart and select "Edit Alt Text". Enter a concise description of the visual’s purpose. For decorative images, check the “Mark as decorative” box.
  • Excel: Right-click a chart, shape, or image and choose "Edit Alt Text". Provide a summary of the data or visual. For decorative visuals, mark as decorative.
  • Outlook: When composing an email, right-click an inserted image and select "Edit Alt Text". Add a meaningful description or mark as decorative.

How to write good alt text

  • Describe essential information or function, in under ~125 characters when possible.
  • Don’t start with “Image of …” — screen readers already announce images.
  • Match the surrounding context; omit irrelevant details.
  • For functional images (e.g., buttons), describe the action (e.g., “Search”).
  • For charts and graphs, summarize key trends or data points.
  • Use plain language and avoid jargon or abbreviations unless explained.

HTML example:

<img src="bar-chart.png" alt="Sales increased 15% from Q1 to Q2">

Decorative images

If an image adds no information, mark it decorative so it’s skipped by screen readers.

HTML: alt="". In authoring tools, use “Mark as decorative.”

Complex graphics and long descriptions

Charts, maps, and infographics may need a longer text alternative. In Microsoft 365, provide concise alt text plus a nearby detailed description in text. Ensure the long description conveys trends, comparisons, and key data points.

<img src="map.png" alt="Map of library branches; see details below.">
<p id="map-desc">There are 12 branches; the newest is downtown, open 7 days a week…</p>

Accessibility tips for non-web documents

  • Describe the function or information conveyed by the image, not just its appearance.
  • Mark purely decorative images so they are ignored by screen readers.
  • Review alt text after exporting to PDF—some tools may not retain alt text correctly.
  • Not updating alt text when images are replaced or updated.

Common mistakes to avoid

  • Leaving alt text blank for informative images.
  • Using file names or generic phrases like “image” or “graphic” as alt text.
  • Failing to mark decorative images appropriately.

Check your work

  • Use your authoring tool’s accessibility checker to find missing alt text.
  • Review with a screen reader; ensure decorative images are skipped.
