# GitHub Project Roadmap - ESDC Accessibility Course Development

This roadmap tracks the development and launch of the ESDC Self-Paced Accessibility Course modules.

**🎯 Key Milestone: English Course Launch - October 26, 2024**

## Project Fields

The following fields should be added to your GitHub Project table:

- **Start date** (type: Date): The date when work begins on the item
- **Target date** (type: Date): The planned completion date for the item
- **Phase** (Single select): Current phase in the development workflow
  - Options: Content Creation, IT Review, Incorporate Comments, Translation, French Integration, Testing/Pilot, Launch
- **Status** (Single select): Current status of the item
  - Options: Not started, In progress, Done
- **Owner** (Text or Single select): Person responsible for the item
  - Options: Chris K, IT Team, Translation Team

## Phases

1. **Content Creation** - Initial content development and writing
2. **IT Review** - Technical review by IT Accessibility Office
3. **Incorporate Comments** - Address feedback and make revisions
4. **Translation** - Send for French translation
5. **French Integration** - Integrate French content into GitHub
6. **Testing/Pilot** - Pilot testing with ITAO team
7. **Launch** - Official course release

## Implementation Guide

### Setting up GitHub Project

1. Create a new GitHub Project (Projects → New Project)
2. Choose "Table" view
3. Add the following custom fields:
   - **Start date** (Date field)
   - **Target date** (Date field)
   - **Phase** (Single select with 7 options: Content Creation, IT Review, Incorporate Comments, Translation, French Integration, Testing/Pilot, Launch)
   - **Status** (Single select with 3 options: Not started, In progress, Done)
   - **Owner** (Text or Single select with options: Chris K, IT Team, Translation Team)

4. Add each roadmap item from the table above as a project item
5. Fill in the fields based on the values in this roadmap
6. Use markers (🚀, 🌐, 📅, ⚠️) in item descriptions to highlight key milestones
