## Using the Roadmap

The roadmap documents provide everything needed to set up a GitHub Project for tracking the ESDC Accessibility Course development:

1. Start with **[ROADMAP-QUICKSTART.md](ROADMAP-QUICKSTART.md)** for a quick overview
2. Refer to **[project-roadmap.md](project-roadmap.md)** for complete details
3. The roadmap includes:
   - 8 project items (6 modules + Site Map + All Best Practices)
   - 7 development phases
   - Start and target dates
   - Status tracking
   - Owner assignments
   - Key milestone markers

### Key Dates

- **October 26, 2024** - English Course Launch Deadline ⚠️
- **October 27, 2024** - English ITAO Pilot Launch 🚀
- **November 24, 2024** - French ITAO Pilot Launch 🚀
