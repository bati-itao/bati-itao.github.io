## Slide 3: Document Properties
- Set meaningful title, author, subject
- Improves searchability
- Supports assistive technologies
- Access through File > Info > Properties
- Export settings preserve properties

## Slide 4: Accessibility Assistant
- Built-in Microsoft 365 feature
- Real-time accessibility guidance
- Turn on through Review > Check Accessibility
- Customizable notifications
- Learning-focused approach

## Slide 5: Language Settings
- Set document language for proper pronunciation
- Review > Language > Set Proofing Language
- Mark foreign language passages
- Critical for screen reader accuracy

## Slide 6: Text and Fonts
- Use readable fonts (Arial, Calibri, Verdana)
- Minimum 12pt size
- Proper line spacing
- Consistent formatting
- Avoid decorative fonts for body text

## Slide 7: Alternative Text
- Describe purpose, not appearance
- Keep concise (under 125 characters)
- Mark decorative images appropriately
- Right-click > Edit Alt Text
- Essential for screen readers

## Slide 8: Color Contrast
- Minimum 4.5:1 ratio for normal text
- 3:1 for large text (18pt+ or 14pt+ bold)
- Don't rely on color alone
- Test with accessibility checker
- Use high contrast combinations

## Slide 9: Accessible Tables
- Use table headers (not just bold text)
- Add table caption/title
- Simple structure preferred
- Avoid merged cells when possible
- Logical reading order

## Slide 10: Acronyms and Abbreviations
- Expand on first use
- Format: "Web Content Accessibility Guidelines (WCAG)"
- Provide glossary for long documents
- Don't use periods in acronyms

## Slide 11: Hyperlinks
- Descriptive link text
- Avoid "click here" or "read more"
- Include file type/size when relevant
- Ensure links are underlined
- Use meaningful context

## Slide 12: Plain Language
- Short sentences and common words
- Active voice
- Clear headings
- Logical structure
- Define technical terms

## Slide 13: Accessible Templates
- Simple, consistent layouts
- Built-in accessibility features
- Clear instructions and examples
- Accessible color schemes
- Proper heading structure
