# GitHub Project Roadmap - Quick Start Guide

## What is this?

This roadmap document (`project-roadmap.md`) provides a comprehensive plan for creating a GitHub Project table to track the ESDC Accessibility Course development.

## How to Use

### Step 1: Create GitHub Project
1. Go to your repository on GitHub
2. Click "Projects" tab
3. Click "New Project"
4. Select "Table" view
5. Give it a name like "ESDC Accessibility Course Roadmap"

### Step 2: Add Custom Fields
Add these fields to your project table:

| Field Name | Field Type | Options/Values |
|------------|-----------|----------------|
| Start date | Date | - |
| Target date | Date | - |
| Phase | Single select | Content Creation, IT Review, Incorporate Comments, Translation, French Integration, Testing/Pilot, Launch |
| Status | Single select | Not started, In progress, Done |
| Owner | Text or Single select | Chris K, IT Team, Translation Team |

### Step 3: Add Items
Copy the 8 items from the roadmap:
- Module 1
- Module 2
- Module 3
- Module 4
- Module 5
- Module 6
- Site Map
- All Best Practices

### Step 4: Fill in Data
For each item, copy the values from the tables in `project-roadmap.md`:
- Start date
- Target date
- Phase
- Status
- Owner

### Step 5: Track Progress
Update the Status field as work progresses:
- **Not started** → **In progress** → **Done**
