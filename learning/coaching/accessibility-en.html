Accessibility coaching

General description

ESDC is committed to ensuring the accessibility of its website and desktop assets. These coaching sessions outline federal standards and guidelines on accessibility and emphasizes the requirements, roles and responsibilities involved in compliance with these standards.

  • Coaching session references
  • Feedback survey form - Accessibility coaching series

Full day hands-on session

Target audience

Developers, programmers, analysts, technical advisors, QA analysts, testers and any other position that requires an in-depth look at accessibility requirements

Technical session covers

  • Educating participants on accessibility through code validation and hands-on testing.
  • Learn about the Accessible Canada Act and how it applies to ESDC
  • ESDC accessibility roadmap
  • In-depth look at the Web Content Accessibility Guidelines (WCAG) 2.1.
  • Learn to use various accessibility evaluation tools that are used at ESDC to verify compliance.

Note: Regional participants need to follow the Software readiness for Full-day Hands-on Coaching series instructions prior to the day of the course.

Half day information session

Target audience

Project managers, project leads, managers, directors and any other position that requires introduction to accessibility in a non-technical manner.

Session covers

  • Educate on accessibility.
  • Learn about the Accessible Canada Act and how it applies to ESDC
  • ESDC accessibility roadmap
  • Introduction on Web Content Accessibility Guidelines (WCAG) 2.1.
