Modules complémentaires de Firefox

Veuillez installer les modules complémentaires suivants dans Firefox

  • WAVE toolbar (disponible seulement en anglais)
  • Complex data table markup tool (boîte de dialogue d'instructions)
  • aXe developer tools (disponible seulement en anglais)
  • Nordburg tools (disponible seulement en anglais)
    • Nordburg Focus Revealer
    • Nordburg Forms Mode Simulator
    • Nordburg Validator
    • SlashOn BW filter
    • SlashOn Tag finder

Complex data table markup tool (boîte de dialogue d'instructions)

Pour installer l'outil, Complex data table markup tool (disponible seulement en anglais), vous devez procéder comme suit :

  • téléchargez le fichier comtab.zip sur votre système,
  • décompressez le fichier
  • ouvrir le fichier, complex_table_toolbar-1.1.7-an+fx.xpi dans Firefox
