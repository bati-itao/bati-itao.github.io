Overview

Proper text structure and reading order are fundamental to accessible presentations. Screen readers rely on logical heading hierarchies and reading order to help users navigate and understand content effectively.

Creating proper heading structure

Slide titles automatically create heading structure when using built-in layouts:

  • Each slide title becomes a Level 1 heading for screen readers
  • Use the built-in title placeholder rather than text boxes for slide titles
  • Make slide titles descriptive and unique when possible
  • Avoid skipping heading levels within slide content
  • Use heading styles for subsections within complex slides

Accessible text formatting

Format text to enhance accessibility:

  • Use meaningful formatting: Bold and italic should convey meaning, not just visual emphasis
  • Avoid color alone: Don't rely solely on color to convey information
  • Use sufficient text size: Minimum 18pt for body text, 24pt for titles
  • Choose readable fonts: Sans-serif fonts like Arial, Calibri, or Verdana
  • Use proper spacing: Adequate line spacing and paragraph spacing

Managing reading order

Ensure logical reading order for screen readers:

  1. Open the Selection Pane (Home tab > Arrange > Selection Pane)
  2. Review the order of objects - screen readers read from bottom to top in the pane
  3. Drag objects to reorder them if needed
  4. Keep related content together in the reading order
  5. Test with keyboard navigation (Tab key) to verify order

Font and readability considerations

Choose fonts and formatting that support readability:

Font families
Use simple, clean fonts without decorative elements. Arial, Calibri, Verdana, and Tahoma are good choices.
Font size
Use 18pt minimum for body text, 24pt for slide titles. Larger text may be needed for presentations viewed from a distance.
Text alignment
Left-align text for better readability. Avoid justified text which can create uneven spacing.
Line spacing
Use 1.2 to 1.5 line spacing for better readability, especially for users with dyslexia.
