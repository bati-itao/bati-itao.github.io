What it is

PowerPoint accessibility means creating presentations that everyone can use. This includes people who:

  • Use screen readers to hear slide content
  • Navigate with keyboard instead of mouse
  • Have vision, hearing, or motor difficulties
  • Use different devices or assistive technology

When you make accessible presentations, you follow specific requirements and use PowerPoint's built-in tools to ensure your content works for everyone.

Key requirements

Based on government accessibility standards, your PowerPoint presentations must:

Structure requirements

  • Have a unique title on each slide that describes the content
  • Use built-in slide layouts with proper placeholders
  • Present content in logical reading order (left to right, top to bottom)
  • Avoid automatic timed animations or slide transitions

Content requirements

  • Provide alt text for all images and graphics
  • Use sufficient color contrast (4.5:1 for normal text, 3:1 for large text)
  • Don't use color alone to show information
  • Include captions or transcripts for videos and audio
  • Make tables with proper headers using built-in styles

Text requirements

  • Allow users to resize text without losing content
  • Set the correct language for all text content
  • Write descriptive link text that works out of context

Built-in tools to help you

PowerPoint 365 has tools that make accessibility easier. Learn to use these features:

Key accessibility tools

Accessibility Checker
Finds accessibility problems and suggests fixes. Access it from Review > Check Accessibility
Reading Order Pane
Shows how screen readers will read your slide. Use it to fix reading order problems
Alt Text pane
Easy way to add descriptions for images and graphics that screen readers can use
Design Ideas
Suggests layouts that keep good structure while improving visual design

Helpful keyboard shortcuts

  • Alt + Shift + F10: Open Reading Order Pane
  • F7: Check spelling
  • Ctrl + K: Add a hyperlink

How to get started

Follow these steps to set up PowerPoint for accessibility:

  1. Turn on the Accessibility Checker:
    • Go to File > Options > Ease of Access
    • Check "Keep accessibility checker running while I work"
    • This will alert you to problems as you create slides
  2. Learn the tools:
    • Find the Review tab on the ribbon
    • Look for "Check Accessibility" button
    • Practice opening the Reading Order Pane
  3. Plan before you build:
    • Think about your main message
    • Consider who will see your presentation
    • Decide on a simple, clear structure

Quick accessibility checklist

Use this checklist for every presentation you create:

Before you start:

  • ☐ Enable Accessibility Checker in settings
  • ☐ Choose a simple, high-contrast template
  • ☐ Plan your slide structure and key messages

While you work:

  • ☐ Use built-in slide layouts (not blank slides)
  • ☐ Add unique titles to every slide
  • ☐ Put alt text on all images
  • ☐ Check color contrast meets standards
  • ☐ Keep reading order logical

Before you share:

  • ☐ Run the Accessibility Checker
  • ☐ Fix all errors and warnings
  • ☐ Test navigation with keyboard only
