Goal

Help creators build PowerPoint decks that are perceivable, operable, understandable, and robust for everyone—including screen reader users, keyboard-only users, and people with low vision, color-vision deficiency, cognitive disabilities, or hearing loss.

PowerPoint accessibility is essential because presentations are a primary means of communication in business, education, and public settings. When presentations are inaccessible, they exclude people with disabilities from participating fully in meetings, training sessions, and important information sharing.

Key outcomes

By the end of this PowerPoint accessibility training, you will be able to:

  • Use correct slide structures (layouts/placeholders) instead of drawing text boxes
  • Ensure every slide has a unique, meaningful title (use hidden titles when necessary)
  • Set logical reading order with the Selection Pane
  • Use accessible themes (fonts, sizes, and contrast) via the Slide Master
  • Provide accessible alternatives for images, charts, SmartArt, and multimedia
  • Avoid problematic animations/transitions; reduce motion and flashing
  • Export and share in ways that retain accessibility wherever possible
  • Present accessibly using live subtitles and descriptive language

WCAG mapping (high level)

The PowerPoint accessibility techniques covered in this module address several Web Content Accessibility Guidelines (WCAG) 2.1 success criteria:

1.1.1 Non-text Content
Providing alternative text for images, charts, and decorative graphics
1.2.x Time-based Media
Adding captions and subtitles to video and audio content
1.3.1 Info and Relationships
Using proper structure via layouts and placeholders instead of visual formatting
1.4.1 Use of Color; 1.4.3 Contrast (Minimum); 1.4.11 Non-text Contrast
Ensuring sufficient color contrast and not relying on color alone
2.2.x Enough Time / Pause, Stop, Hide
Controlling animations and automatic slide transitions
2.3.1 Three Flashes or Below Threshold
Avoiding flashing content that could trigger seizures
2.4.6 Headings and Labels
Using slide titles as navigational labels
4.1.x Compatibility
Ensuring proper object names and reading order for assistive technologies
