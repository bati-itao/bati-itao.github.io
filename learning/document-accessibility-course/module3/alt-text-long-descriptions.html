What it is

Alt text (alternative text) is a short description of an image that screen readers can read aloud. When someone can't see an image, the alt text tells them what the image shows and why it's important.

Long descriptions are longer explanations for complex images like charts, graphs, or detailed diagrams. They provide more detail than alt text can include.

In PowerPoint, you add alt text through the Alt Text pane. This is an accessibility requirement for all images that aren't purely decorative.

Why alt text matters

Alt text helps many people, including:

  • Screen reader users: People who are blind or have vision difficulties
  • People with cognitive disabilities: Who process text better than complex images
  • Voice interface users: Who navigate content by voice
  • Everyone: In situations where images can't be displayed

Real example

A chart showing "Q4 sales increased 23%" is crucial information. Without alt text, screen reader users only hear "image" and miss this key data.

With good alt text: "Bar chart showing Q4 sales increased 23% from $2.1M to $2.6M, exceeding target of $2.4M."

How to add alt text

Follow these steps to add alt text to any image in PowerPoint:

  1. Select the image:
    • Click on the image you want to describe
    • The image should show selection handles around it
  2. Open the Alt Text panel:
    • Right-click the image and choose "Edit Alt Text"
    • OR use the ribbon: Picture Format > Alt Text
    • The Alt Text panel will open on the right side
  3. Write your description:
    • Type your description in the text box
    • Keep it under 140 characters for English (170 for French)
    • Focus on the most important information
  4. For decorative images:
    • Check the "Mark as decorative" box
    • Leave the text field empty
    • Screen readers will skip these images

✅ PowerPoint's auto-suggestions

PowerPoint 365 can suggest alt text automatically:

  • You'll see suggested text when you open the Alt Text panel
  • Always review and edit these suggestions
  • Auto-generated text often misses context or key details
  • Use suggestions as a starting point, not the final answer

How to write good alt text

Good alt text follows these best practices from government accessibility standards:

  1. Be concise but complete:
    • Keep text under 140 characters when possible
    • Include the most important information first
    • Don't start with "Image of" or "Picture of"
  2. Focus on function and meaning:
    • Describe what the image shows that's relevant to your content
    • Explain why the image is there
    • Include key data or findings from charts and graphs
  3. Use plain language:
    • Use familiar words your audience will understand
    • Avoid technical jargon unless necessary
    • Be specific and concrete
  4. Match the context:
    • Consider what's already said in the slide text
    • Don't repeat information that's already available
    • Add information that's only in the image

Alt text examples

❌ Poor alt text:

  • "Chart"
  • "Image of data"
  • "Screenshot"
  • "Graph showing results"

✅ Good alt text:

  • "Sales increased 15% in Q3"
  • "Login screen with username and password fields"
  • "Team of 5 people in office meeting"
  • "Pie chart: 60% mobile users, 40% desktop users"

How to handle decorative images

Decorative images are used for visual appeal but don't add essential information. Here's how to handle them:

  1. Identify decorative images:
    • Background patterns or textures
    • Stock photos used for mood (not specific information)
    • Decorative borders or dividers
    • Company logos used for recognition (when context is clear)
  2. Mark them as decorative:
    • Right-click the image and choose "Edit Alt Text"
    • Check the "Mark as decorative" box
    • Leave the description field empty
    • Screen readers will skip these images completely
  3. When not to mark as decorative:
    • If the image shows something specific to your content
    • If removing the image would make the slide confusing
    • If the image helps people understand your message
    • When in doubt, add brief alt text rather than mark as decorative

⚠️ Be careful with decorative marking

Only mark images as decorative if they truly add no information. For example:

  • Don't mark as decorative: A photo of your team, even if it's just for visual appeal
  • Do mark as decorative: A generic pattern used as a background

How to describe complex images

Complex images like charts, graphs, and detailed diagrams need more than simple alt text. Here's how to handle them following best practices:

  1. Start with short alt text:
    • Add brief alt text that identifies the type of image
    • Include the main finding or conclusion
    • Example: "Bar chart showing 25% increase in website traffic"
  2. Add long descriptions when needed:
    • Use speaker notes for detailed descriptions
    • Add a separate slide with the data table
    • Include key data points in the slide text
    • Reference the data in your verbal presentation
  3. For charts and graphs:
    • Alt text: chart type + key finding
    • Long description: data trends, highest/lowest values, comparisons
    • Consider including a data table on the same or next slide
  4. For screenshots and diagrams:
    • Alt text: what the image shows + main purpose
    • Long description: step-by-step process or key elements
    • Use bullet points in slide text to list key steps

Complex image examples

Organization Chart:

  • Alt text: "Organization chart showing IT department structure"
  • Long description: "IT Director at top, with three managers below: Security, Development, and Support. Each manager has 3-4 team members reporting to them."

Sales Chart:

  • Alt text: "Line graph showing quarterly sales growth exceeded targets"
  • Long description: "Q1: $1.2M actual vs $1M target. Q2: $1.5M actual vs $1.2M target. Q3: $1.8M actual vs $1.4M target. Consistent upward trend with increasing gap above targets."

Examples and practice

Here are real-world examples of different image types and their alt text:

Charts and data visualizations

Pie chart

Alt text: "Pie chart: Mobile users 65%, desktop 25%, tablet 10%"

Bar chart

Alt text: "Customer satisfaction increased from 3.2 to 4.1 stars over 6 months"

Screenshots and interfaces

Software screenshot

Alt text: "PowerPoint ribbon showing Review tab with Check Accessibility button highlighted"

Process diagram

Alt text: "Three-step process: Plan, Execute, Review, with arrows connecting each step"

Photos and illustrations

Team photo

Alt text: "Marketing team of 6 people posing in conference room"

Product image

Alt text: "New laptop model showing thin profile and backlit keyboard"

✅ Alt text checklist

Before finishing your presentation, check that:

  • ☐ Every image has alt text or is marked as decorative
  • ☐ Alt text describes the image's purpose, not just appearance
  • ☐ Complex images have additional description in slide text or notes
  • ☐ Alt text is concise (under 140 characters when possible)
  • ☐ Alt text uses plain language your audience will understand
  • ☐ Charts include key data and findings in alt text
  • ☐ Decorative images are properly marked
