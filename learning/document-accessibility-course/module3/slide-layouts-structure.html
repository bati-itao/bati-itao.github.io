What it is

Slide layouts and structure are about organizing your content so screen readers and other assistive technology can understand it. Good structure means using PowerPoint's built-in layouts instead of blank slides, putting content in the right placeholders, and creating titles that make sense.

When you use proper structure, people using screen readers can navigate your slides easily, skip to the content they want, and understand how information relates to each other.

Why structure matters

Proper slide structure follows government accessibility requirements and provides these benefits:

  • Screen reader navigation: Users can jump between slides and find content quickly
  • Logical reading order: Content is read in the right sequence
  • Clear relationships: Titles, headings, and content connect properly
  • Keyboard navigation: Tab key moves through content in logical order
  • Better for everyone: Organized content is easier for all users to follow

❌ Avoid these structure problems

  • Using blank slides (no built-in structure)
  • Adding text boxes instead of using placeholders
  • Slides without titles
  • Content that doesn't follow logical order
  • Mixing different types of content without clear organization

How to choose layouts

Follow these steps to pick the right layout for your content:

  1. Match layout to content type:
    • Title Slide: For presentation opening and section dividers
    • Title and Content: Most common - use for text, images, or mixed content
    • Two Content: For comparisons or side-by-side information
    • Section Header: To divide presentation into major sections
    • Content with Caption: For large images with explanatory text
  2. Apply the layout:
    • Select the slide you want to change
    • Go to Home tab > Layout button
    • Choose the layout that fits your content
    • PowerPoint will move existing content to the right placeholders
  3. Check the result:
    • Make sure content moved to appropriate areas
    • Add a title if the slide doesn't have one
    • Adjust content as needed within placeholders

✅ Quick layout selection guide

  • Text only: Title and Content
  • Image only: Title and Content or Content with Caption
  • Comparing two things: Two Content or Comparison
  • Data visualization: Title and Content
  • Starting new section: Section Header

How to use placeholders

Placeholders are pre-made areas for different types of content. Here's how to use them correctly:

  1. Always use the title placeholder:
    • Click in the title area at the top of each slide
    • Add a unique, descriptive title
    • Never leave title placeholders empty
    • Don't delete title placeholders
  2. Use content placeholders for all body content:
    • Click in the content area to add text, images, or other content
    • Don't add separate text boxes - use the placeholder instead
    • Content placeholders handle multiple types: text, images, charts
  3. Match placeholder type to content:
    • Use picture placeholders for images when available
    • Use chart placeholders for data visualizations
    • Use table placeholders for data tables
    • General content placeholders work for mixed content
  4. Don't delete unused placeholders:
    • Leave empty placeholders as they are
    • Choose a different layout if you need fewer content areas
    • Deleting placeholders can break the accessibility structure

⚠️ Common placeholder mistakes

  • Adding text boxes instead of using content placeholders
  • Leaving slides without titles
  • Deleting placeholders instead of choosing better layouts
  • Putting the wrong content type in specialized placeholders

How to create slide titles

Every slide must have a unique, descriptive title. Here's how to write good ones:

  1. Write descriptive titles:
    • Describe what the slide is about
    • Be specific, not generic
    • Use 5-10 words when possible
    • Make each title unique
  2. Use clear, simple language:
    • Use familiar words your audience will understand
    • Avoid jargon and abbreviations
    • Use active voice when possible
  3. Be consistent:
    • Use similar formats for related slides
    • Use parallel structure (same grammar pattern)
    • Add prefixes for series: "Step 1:", "Module 2:", etc.
  4. Handle visual design needs:
    • If you need to hide titles visually, don't delete them
    • Move hidden titles off-screen but keep them in the placeholder
    • Use subtitle areas for visual impact while keeping hidden titles

Examples of good vs. poor titles

❌ Poor titles:

  • Introduction
  • Data
  • Next Steps
  • Questions

✅ Good titles:

  • Welcome to Customer Service Training
  • Q3 Sales Results by Region
  • Action Items and Timeline
  • Q&A and Contact Information

Common problems and fixes

Here are solutions to the most common layout and structure problems:

Problem: Content doesn't fit in placeholders

  • Solution: Choose a different layout that fits your content better
  • Solution: Break content into multiple slides
  • Solution: Use bullet points instead of long paragraphs
  • Don't: Add text boxes outside placeholders

Problem: Slide titles look repetitive

  • Solution: Add context or numbers: "Step 1: Planning", "Step 2: Design"
  • Solution: Use descriptive phrases: "Budget Overview", "Budget Details", "Budget Timeline"
  • Solution: Hide titles visually but keep them for accessibility

Problem: Reading order is wrong

  • Solution: Use built-in layouts instead of blank slides
  • Solution: Don't add content outside placeholders
  • Solution: Use the Reading Order pane to check and fix order

Problem: Accessibility Checker shows layout errors

  • Missing slide titles: Add a title to every slide
  • Reading order issues: Use placeholders, don't add independent text boxes
  • Empty placeholders: Either use the placeholder or choose a different layout

✅ Quick structure checklist

Before moving to the next slide:

  • ☐ Slide has a unique, descriptive title
  • ☐ Used built-in layout (not blank slide)
  • ☐ Content is in placeholders, not separate text boxes
  • ☐ Reading order makes sense (check with Tab key)
  • ☐ Layout matches the type of content

Using headings effectively

  • Slide title = H1: The slide title serves as the main heading
  • Major sections = H2: Use for main content sections within a slide
  • Subsections = H3: For sub-topics under major sections
  • Avoid deep nesting: Rarely go beyond H3 on a single slide

Creating headings in PowerPoint 365

  1. Select the text you want to make a heading
  2. Go to Home > Styles
  3. Choose the appropriate heading level
  4. Or use the outline view to organize content hierarchically

Visual hierarchy best practices

  • Size relationships: Larger text for more important headings
  • Color consistency: Use consistent colors for similar heading levels
  • White space: More space around higher-level headings
  • Font weight: Bold for headings, regular for body text

✅ Testing your hierarchy

To test if your hierarchy is clear:

  1. Switch to View > Outline View
  2. Look at the text-only outline - does it make sense?
  3. Use Tab key to navigate through the slide - is the order logical?
  4. Ask someone else to identify the main points from the visual hierarchy

Common accessible layout patterns

Here are some layout patterns that work well for accessibility while maintaining visual appeal.

Text-heavy slides

  • Use "Title and Content" layout
  • Break up text with:
    • Bulleted or numbered lists
    • Short paragraphs (3-4 sentences max)
    • Subheadings to organize topics
    • White space for visual breathing room

Image-focused slides

  • Use "Content with Caption" layout
  • Always include:
    • Descriptive alt text for the image
    • Caption text that explains the image's relevance
    • Context about why the image is important

Data presentation slides

  • Use "Title and Content" with chart placeholder
  • Include:
    • Clear chart title describing the data
    • Alt text summarizing key findings
    • Data table as backup (can be hidden but available)
    • Explanation of trends or important points

Comparison slides

  • Use "Comparison" or "Two Content" layout
  • Structure with:
    • Clear headings for each side of the comparison
    • Parallel structure in content organization
    • Consistent formatting on both sides
    • Summary of key differences or similarities

Troubleshooting layout issues

Common problems and solutions

Content doesn't fit in placeholders

  • Try a different layout: Two Content might work better than Title and Content
  • Reduce content: Consider splitting into multiple slides
  • Adjust font size: But ensure it remains readable (18pt minimum)
  • Use outline format: Bulleted lists take up less space than paragraphs

Reading order seems wrong

  • Check with Tab key: Navigate through the slide to see the order
  • Use Reading Order Pane: View > Reading Order Pane to see and adjust order
  • Simplify the layout: Complex layouts often have order issues
  • Group related elements: Use PowerPoint's grouping feature

Accessibility Checker flags layout issues

  • Missing slide title: Always add a descriptive title
  • Reading order problems: Use the Reading Order Pane to fix
  • Text outside placeholders: Move content into appropriate placeholders
  • Empty placeholders: Remove unused ones or choose a different layout

Quick layout checklist

  • ☐ Used a built-in layout (not blank slide)
  • ☐ Every slide has a unique, descriptive title
  • ☐ Content is in placeholders, not floating text boxes
  • ☐ Reading order flows logically (test with Tab key)
  • ☐ Visual hierarchy is clear and consistent
  • ☐ Layout serves the content, not just visual design
  • ☐ No accessibility checker errors related to structure
