Accessible Templates

The IT Accessibility Office (ITAO) has created an accessible PowerPoint Template available to use. It will start you on the right path!

Steps to use the template:

  1. Open the ESDC Accessible PowerPoint Template
  2. Navigate to and select “File” from the menu ribbon
  3. Navigate to and select “Save as”
  4. Select the current file type “PowerPoint Template (*.potx)”, and change it to “PowerPoint Presentation (*.pptx)”
  5. Select your preferred save location and select the “Save” button

Important! An accessible template will start you on the right path. However, accessibility verifications are still required to ensure no barriers are introduced.

Finding Accessible Templates

If you need a different layout, you can search for the term “Accessible” in the New Presentation dialogue box in PowerPoint to find other accessible templates.

Templates are a great starting point for accessibility, but the rest of this module will help explain how to ensure the template remains accessible.
