Module 3 – PowerPoint Accessibility

What it is

PowerPoint accessibility means creating presentations that work for everyone. This includes people who use screen readers, have vision or hearing difficulties, or use keyboard navigation instead of a mouse.

This module teaches you to use PowerPoint 365's built-in tools to make presentations that meet accessibility standards. You'll learn to create content that's clear, well-structured, and works with assistive technology.

Why it matters

  • Legal requirement: Government content must meet accessibility standards
  • Better for everyone: Clear structure and design helps all users
  • Professional standard: Shows attention to quality and inclusion
  • Wider reach: More people can use and share your content

How to use this module

This module works best when you follow the lessons in order. Each lesson builds on the previous one:

  1. Start with templates – Learn to choose and set up accessible templates
  2. Build structure – Use proper slide layouts and organize content
  3. Add content – Create accessible text, images, tables, and media
  4. Test and share – Check your work and export accessible presentations

Time needed: About 2-3 hours to complete all lessons
