Module 3 – PowerPoint Accessibility

This module teaches you to use PowerPoint 365's built-in tools to make presentations that meet accessibility standards. You'll learn to create content that's clear, well-structured, and works well with assistive technology.

What you'll learn

By the end of this module, you'll be able to:

  • Create proper slide structure with accessible layouts and reading order
  • Create accessible charts
  • Create accessible multimedia content
  • Deliver accessible presentations with PowerPoint Live

Lessons in this module

  1. Slide Master & Templates
  2. Slide Layouts & Structure
  3. Reading Order & Navigation
  4. Alt Text & Long Descriptions
  5. Tables, Charts & Graphs
  6. Multimedia & Animations
  7. Multimedia Captions & Subtitles
  8. Presenting Accessibly
  9. PowerPoint Live & Exporting to PDF
  10. Resources & Support
