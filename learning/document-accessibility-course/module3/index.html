Module 3 – PowerPoint Accessibility

This module teaches you to use PowerPoint 365's built-in tools to make presentations that meet accessibility standards. You'll learn to create content that's clear, well-structured, and works with assistive technology for people who use screen readers, have vision or hearing difficulties, or use keyboard navigation instead of a mouse.

Lessons in this module

  1. Overview
  2. Getting Started with Templates
  3. Slide Layouts & Structure
  4. Reading Order & Navigation
  5. Alt Text & Long Descriptions
  6. Tables, Charts & Graphs
  7. Multimedia & Animations
  8. Presenting & Exporting
  9. Resources & Support
