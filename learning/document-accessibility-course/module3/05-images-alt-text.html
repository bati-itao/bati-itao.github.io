Overview

Alternative text (alt text) provides text descriptions of images for users who cannot see them, including people using screen readers. Proper alt text ensures visual content is accessible to all users.

Writing effective alt text

Good alt text is:

  • Concise: Usually 125 characters or less
  • Descriptive: Describes the content and function of the image
  • Contextual: Relates to the surrounding content and presentation purpose
  • Meaningful: Conveys the same information the image provides visually

Avoid phrases like "image of" or "picture of" - screen readers already announce that it's an image.

Decorative images

Images that are purely decorative should be marked as decorative:

  • Mark as decorative in PowerPoint's Alt Text pane
  • Decorative images don't convey essential information
  • Examples: borders, background patterns, purely aesthetic elements
  • When in doubt, provide alt text rather than marking as decorative

Complex images and charts

For complex visuals, provide both alt text and additional description:

  • Alt text: Brief description of the image type and main point
  • Long description: Detailed explanation in the slide content or speaker notes
  • Data tables: Consider providing data in table format as well as charts
  • Diagrams: Describe relationships and key information textually

Adding alt text in PowerPoint

Multiple ways to add alt text:

  1. Right-click method: Right-click image > Edit Alt Text
  2. Format menu: Select image > Picture Format > Alt Text
  3. Accessibility pane: Review > Check Accessibility > Alt Text
  4. Generated alt text: Review and edit PowerPoint's automatic suggestions
