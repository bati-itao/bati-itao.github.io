Table resources in the Web Experience Toolkit (WET)

The WET style guide for tables describes the different visual and interactive effects of WET CSS class names.

The DataTables plugin into WET provides searching, sorting, filtering, pagination and other advanced features for tables.

The WET Table Validator detects errors in a table’s markup and generates id/headers markup. The generated markup doesn’t carry over existing attributes in the table, which have to be restored to the valid table manually. An alternative to the WET Table Validator is the Complex Data Table Markup Toolbar Firefox extension.
