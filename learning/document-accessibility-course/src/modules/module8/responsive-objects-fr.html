Objets et modules d’extension adaptés

La balise <objet> définit un conteneur pour une ressource externe. Il peut s’agir d’une page Web, d’une photo, d’un lecteur média ou d’une application d’extension.

Il est préférable d’utiliser la balise <img> pour intégrer une image, la balise <iframe> pour intégrer le langage HTML et les balises <video> et <audio> pour intégrer le contenu média.

Les objets et les modules d’extension peuvent être redimensionnés à mesure que la taille de la zone d’affichage change afin de ne pas déborder de la zone d’affichage. Cela peut être fait en réglant CSS max-width="100%" sur le conteneur. Si l’objet ou le module d’extension permet le redimensionnement, il peut remplir complètement le conteneur.

La propriété de max-width n’a aucun effet si le contenu est plus petit que la largeur maximale.

La balise <object> a été conçue à l’origine pour intégrer les modules d’extension du navigateur. Les modules d’extension sont des programmes informatiques qui élargissent les fonctions standard du navigateur. Les modules d’extension ont été utilisés à de nombreuses fins :

  • Exécutez les applets Java (qui ne sont plus pris en charge)
  • Exécutez les commandes ActiveX (qui ne sont plus prises en charge)
  • Affichez les animations Flash (qui ne sont plus prises en charge)
  • Affichez les cartes
  • Faire une analyse de détection de virus
  • Vérifiez un identifiant bancaire
