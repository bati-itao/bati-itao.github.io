Language: Set the document language

Why language settings matter

  • Screen readers rely on language metadata for pronunciation.
  • Spellcheck and hyphenation use the language setting.
  • Search, translation, and voice assistants improve with correct language.

Set the document language

  • HTML: Set the lang attribute on <html>. 
    <html lang="en">...</html>
  • Word/PowerPoint: Review or Set Proofing Language for the whole document.
  • Google Docs: File → Language → choose the document language.
  • PDF (from source): Ensure the source document’s language is set before export.

Mark passages in other languages

  • HTML: Wrap spans with lang. 
    <p>We said <span lang="fr">bonjour</span> to our guests.</p>
  • Word/Google Docs: Select the phrase → set language for that selection.
  • PowerPoint: Set language per text box or selection.

Do not mark brand names or proper nouns as different language unless pronunciation differs substantially.

Check your work

  • Run your tool’s accessibility checker and verify language settings.
  • Test with a screen reader (e.g., NVDA/VoiceOver) to hear pronunciation on foreign words.
