Module 2 : Créer des documents accessibles

Ce module vous enseigne les techniques d'accessibilité essentielles qui s'appliquent à toutes les applications Microsoft 365. Vous apprendrez comment configurer les propriétés de document accessibles, choisir des polices et formatage appropriés, créer des textes alternatifs significatifs et assurer un contraste de couleur adéquat. Ces compétences fondamentales vous aideront à créer des documents accessibles dès le début de votre processus de conception.
