Module 2: Creating Accessible Documents

This module introduces essential accessibility techniques that apply across Microsoft 365 Word, PowerPoint, and Outlook applications. You’ll learn how to set up accessible document properties, choose appropriate fonts and formatting, create meaningful alternative text, and ensure proper color contrast. These foundational skills are a starting point—they help you build documents that are accessible from the outset. However, each Microsoft 365 application has its own specific accessibility features and techniques. To create truly inclusive content, it's important to continue exploring application-specific guidance beyond this module.
