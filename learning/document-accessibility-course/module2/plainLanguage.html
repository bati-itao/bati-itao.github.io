Plain language principles

  • Use short sentences and common words.
  • Use active voice and direct instructions.
  • Front-load key information and avoid jargon.
  • Break up dense text with headings and lists.

Meaningful link text

  • Write link text that describes the destination or action.
  • Avoid “click here” or “read more.”
  • Include file type/size where relevant (e.g., “Annual Report (PDF, 2 MB)”).
<p>Download the <a href="report.pdf">2024 Annual Report (PDF, 2 MB)</a>.</p>

Structure for understanding

  • Use headings (H1–H3) to organize ideas logically.
  • Use lists for steps and sets of related items.
  • Define terms when you first use them.

Test with real users and tools

  • Run a readability check; aim for a reasonable grade level for the audience.
  • Ask peers to scan and summarize your content—if they can’t, refine it.
