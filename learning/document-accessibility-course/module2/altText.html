What is alt text?

Alt text (alternative text) is a short, descriptive alternative presented to users who cannot see an image, graphic, table, or other visual type object. Well written alt text conveys the purpose and content of these objects, ensuring everyone can access the information, regardless of ability. Good alt text has the added benefit of improving discoverability, and usability.

How to add alt text: Step-by-Step

Follow these detailed instructions to add effective alt text to images in your documents:

Microsoft Word (Desktop)

  1. Insert or select the image you want to add alt text to
  2. Right-click on the image
  3. Choose Edit Alt Text from the context menu
  4. In the Alt Text pane that appears on the right:
    • Type a brief, clear description in the text box
    • Keep it under 125 characters (with spaces)
    • Focus on the image object's purpose or key information
  5. If the image is purely decorative, check Mark as decorative
  6. Click outside the pane or press Enter to save

Microsoft PowerPoint (Desktop)

  1. Select the image, chart, or shape on your slide
  2. Right-click and select Edit Alt Text
  3. Alternatively, go to the Picture Format tab and click Alt Text
  4. In the Alt Text pane:
    • Write a concise description that explains the visual's purpose
    • For charts, summarize the key data or trend
    • For decorative images, check Mark as decorative
  5. Close the pane when finished

Alt text and Microsoft Excel

Alt text can behave differently in Excel, so follow these best practices to ensure no alt text is skipped over.

Place images in the cell

Always anchor images to a cell rather than leaving them floating above the grid. Anchored images behave predictably with keyboard navigation and maintain a logical reading order.

Alt Attributes Pane

While alt attributes can be skipped by some screen readers, its still important to use the Alt Text pane. This is the official Microsoft method for accessibility compliance. Also, some screen readers may not announce Alt Text for images inside tables or charts. To ensure everyone gets the information:

  • Add Alt Text in the Alt Text pane for compliance and users who navigate by object.
  • Include a short description in an adjacent cell (or in cell A1 for context) for compatibility with all assistive technologies.

Example of Good Alt Text

“Bar chart showing quarterly sales growth: Q1 – $10K, Q2 – $15K, Q3 – $20K, Q4 – $25K.”

Tip: If the image is decorative and adds no meaningful information, mark it as decorative in the Alt Text pane or remove it from the reading order.

Microsoft Outlook

  1. When composing an email, insert an image using Insert > Pictures
  2. Right-click on the inserted image
  3. Choose Edit Alt Text
  4. Enter a description that helps recipients understand:
    • What the image shows
    • Why it's relevant to your email
    • Any important details for context
  5. For decorative images (like logos), check Mark as decorative
  6. Click outside to save and continue composing

Web versions (Microsoft 365 Online)

  1. Select your image in Word Online, PowerPoint Online, etc.
  2. Look for the Alt Text button in the toolbar, or
  3. Right-click and choose Alt Text if available
  4. Enter your description in the text field
  5. Use the decorative option if the image is purely visual
  6. Click OK or press Enter to save

How to write good alt text

  • Describe essential information or function, in under ~125 characters (with spaces) when possible.
  • Don’t start with “Image of …” — screen readers already announce images.
  • Match the surrounding context; omit irrelevant details.
  • Referential Alt Text: When surrounding text/captions fully convey an image's content, alt text can be shorter and referential (e.g., "Chart showing enrollment data mentioned above").
  • For functional images (e.g., buttons), describe the action (e.g., “Search”).
  • For charts and graphs, summarize key trends or data points.
  • Use plain language and avoid jargon or abbreviations unless explained.

Decorative images

If an image adds no information, mark it decorative so it’s skipped by screen readers.

In authoring tools, use “Mark as decorative.”

Logo Guidance: Logos that serve a function (brand identification) typically should not be marked decorative in formal templates. Provide appropriate alt text describing their purpose.

Complex graphics and long descriptions

Charts, maps, and infographics may need a longer text alternative. In Microsoft 365, provide concise alt text plus a nearby detailed description in text. Ensure the long description conveys trends, comparisons, and key data points.

Common mistakes to avoid

  • Leaving alt text blank for informative images.
  • Using file names or generic phrases like “image” or “graphic” as alt text.
  • Failing to mark decorative images appropriately.

Check your work

  • Use your authoring tool’s accessibility checker to find missing alt text.
  • Review with a screen reader; ensure decorative images are skipped.
