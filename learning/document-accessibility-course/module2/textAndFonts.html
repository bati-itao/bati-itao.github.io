Accessible text and font choices help everyone access and understand your content. This section explains how to choose fonts, adjust spacing, and apply formatting that supports accessibility in documents and presentations.

Text Formatting

  • Using 18+ point minimum font size for virtual presentations and 30+ point for conference rooms or other live in-person presentations.
  • Using a consistent font size throughout.
  • Using a limited number of font variations (max. 3)
  • Using sans serif fonts like Arial, Calibri or Aptos
  • Avoiding light versions of fonts
  • Using consistent font types and sizes for headings and body text

Paragraph Formatting

  1. Use 1.5 line spacing to make text easier to follow.
  2. Align text to the left to keep spacing consistent.
  3. Avoid full justification, which can create uneven gaps (or rivers) between words.
  4. Avoid pressing Enter multiple times to add space, adjust paragraph spacing using the formatting tools.

How to apply accessible text and font settings

Font setting are available in the Home menu in all Microsoft Office applications. Follow the steps above to set up your document or presentation with accessible font and paragraph settings.

Checking Your Work

  • Use the Accessibility Assistant to check accessibility and readability.
  • Increase font size and adjust line spacing as needed.
