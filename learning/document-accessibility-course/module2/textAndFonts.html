Text, fonts, and spacing

Accessible digital content begins with clear, readable text. This section explores best practices for fonts, spacing, and emphasis to ensure documents and presentations are easy to read for everyone. It covers how to choose appropriate font styles and sizes, apply consistent formatting, and avoid common readability issues.

Fonts and Size

Choose common, readable fonts such as Aptos, Arial, or Calibri. For body text, use at least 12 pt in documents and 18–24+ pt for slide content. Avoid decorative fonts, ALL CAPS, and italics (except in legal documents), as these reduce readability.

Line Spacing and Alignment

Proper spacing and alignment make text easier to read. Use 1.5 line spacing for most content. Align text to the left and avoid full justification, which can create uneven spacing. Maintain consistent paragraph spacing without adding extra blank lines.

Emphasis and Color

Do not rely on color alone to convey meaning—combine it with text or icons (see the contrast lesson). Apply emphasis consistently, using styles like bold or underline sparingly and purposefully.

How to Apply Accessible Text and Font Settings

Follow these steps to set up accessible text formatting in your documents and presentations.

Setting Default Fonts in Microsoft Word

  1. Open a new document in Microsoft Word.
  2. Go to the Home tab.
  3. Click the arrow in the bottom-right corner of the Font section.
  4. In the Font dialog box:
    • Select an accessible font: Aptos, Arial, or Calibri.
    • Set size to at least 11 pt for body text.
    • Ensure style is set to “Regular” (not “Light” or “Thin”).
  5. Click Set As Default.
  6. Choose All documents based on the Normal template.
  7. Click OK.

Adjusting Line Spacing for Better Readability

  1. Select all text with Ctrl+A (Windows) or Cmd+A (Mac).
  2. Go to Home → Paragraph section.
  3. Click the Line Spacing button.
  4. Choose 1.5 for optimal readability, or:
  5. Select Line Spacing Options for custom settings:
    • Set “Line spacing” to “Multiple.”
    • Enter 1.2 to 1.5 in the “At” field.
    • Click OK.

Setting Up Accessible Heading Styles

  1. Go to Home → Styles section.
  2. Right-click on Heading 1 and select Modify.
  3. In the Modify Style dialog:
    • Choose a clear font (same as body text for consistency).
    • Set size to 18–20 pt for Heading 1.
    • Make it Bold.
    • Choose a high-contrast color (dark blue or black).
  4. Click OK.
  5. Repeat for Heading 2 (16 pt) and Heading 3 (14 pt), making each smaller than the previous.

Creating Accessible Text Formatting in PowerPoint

  1. Open PowerPoint and create a new presentation.
  2. Go to View → Slide Master.
  3. Select the master slide (first large one).
  4. For the title placeholder:
    • Select the title text.
    • Choose font: Aptos, Arial, or Calibri.
    • Set size to at least 28 pt.
    • Make it bold.
  5. For content placeholders:
    • Select the content text.
    • Use the same font family.
    • Set size to at least 24 pt for presentations.
    • Ensure adequate line spacing.
  6. Click Close Master View when finished.

Checking and fixing font issues

  1. Run the accessibility checker:
    • Go to Review → Check Accessibility
    • Look for warnings about hard-to-read text
    • Click on issues to see specific problems
  2. Test readability:
    • Zoom to 200% using Ctrl + Plus
    • Check that text remains readable without horizontal scrolling
    • Print preview to test how text appears on paper
  3. Fix common problems:
    • Replace decorative fonts in body text with Aptos, Arial, Calibri
    • Increase font size if smaller than 12pt
    • Change light or thin fonts to regular weight
    • Add spacing between lines if text looks cramped

Setting paragraph spacing instead of extra lines

  1. Select the paragraph you want to adjust
  2. Right-click and choose Paragraph
  3. In the Paragraph dialog:
    • Set "Spacing Before" to 6pt or 12pt
    • Set "Spacing After" to 6pt or 12pt
    • This replaces the need for blank lines
  4. Click OK
  5. Apply this setting to your styles for consistency

Check your work

  • Zoom to 200% to confirm content remains legible.
  • Print preview or export to PDF to check spacing consistency.
  • PowerPoint Tip: Use Slide Master to enforce consistent, accessible fonts, sizes, and spacing across layouts.
