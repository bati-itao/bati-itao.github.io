Hyperlinks: Make links clear and useful

What you will learn

You will learn how to write link text that is easy to scan and easy to understand. This helps everyone, especially people who use a keyboard or a screen reader.

Why clear link text matters

Good links tell people exactly what they will get. Clear link text saves time, reduces confusion, and helps people decide if they should follow the link. It also helps people who scan a list of links without the surrounding paragraph.

How to write links

  • Use short, descriptive text that names the action or the destination. Example: “Apply for travel reimbursement”.
  • Avoid “click here”, “read more”, or pasting long web addresses. These do not explain the purpose.
  • Put the link on the words that matter, not on a whole sentence.
  • If the link opens a file, say so when it helps. Example: “Quarterly results (PDF)”.
  • If you have several similar links, make each one specific. Example: “Benefits overview” and “Benefits dates”.

Give helpful context

Place links close to the information they relate to. Add a short line before the link if you need to explain why the link is useful. Avoid writing “the link on the right” or other visual directions.

Linking to files

  • Use clear file names and link text so people know what the file is.
  • Mention the file type and size when it could affect someone’s choice to download.
  • Share the main, up-to-date location when possible, not copies that may go out of date.

Avoid common mistakes

  • Replace “click here” with the action: “Complete security training”.
  • Turn bare URLs into descriptive links. Keep bare URLs only when people need to copy the address.
  • Do not use the same vague link text for different destinations.
  • Make sure links look like links. Use an underline or another clear visual cue.

Quick check

  • Read the links on the page out loud, one by one. Do they make sense on their own?
  • Tab through the links. Is the order logical? Can you see where focus is?
  • Are links easy to see next to body text?
