Why the Properties Pane matters

The Properties pane in Microsoft Word, PowerPoint, Excel, and Outlook lets you set key details about your file. These properties—like Title, Author, Subject, and Keywords—help users find, identify, and use your documents. They also support accessibility by providing screen readers and other assistive technologies with important context.

Where to find the Properties pane

  • Word, PowerPoint, Excel: Go to File > Info to view and edit properties.
  • Outlook: For attachments, right-click the file and choose Properties . Note that attachment properties are inherited from the source file and many properties are read-only in Outlook.

The Properties pane is available in both desktop and web versions, though options may vary.

How to access and update document properties

Follow these step-by-step instructions to set up your document properties for better accessibility and discoverability:

In Word, PowerPoint, or Excel (Desktop)

  1. Open your document
  2. Click File in the top menu
  3. Click Info in the left panel
  4. Look for the Properties section on the right side of the screen
  5. Click Properties dropdown and select Advanced Properties for more options

In Microsoft 365 Web Apps

  1. Open your document in the web browser
  2. Click File in the top menu
  3. Select Info from the menu
  4. Find the Properties panel on the right side
  5. Click on any property field to edit it directly

Step-by-step property updates

  1. Set the Title: Click in the Title field and enter a clear, descriptive name (e.g., "Employee Handbook 2024" instead of "doc_final_v3")
  2. Add the Subject: Write a brief summary of what the document contains (e.g., "Company policies and procedures for new employees")
  3. Update Author: Enter your name or your team/department name
  4. Add Keywords: Type 3-5 search terms separated by commas (e.g., "policies, handbook, employees, procedures")
  5. Save your changes: Click outside the property fields or press Enter to save

Key properties to fill in

  • Title: Use a clear, descriptive name for the document. Avoid codes or internal jargon.
  • Subject: Add a brief summary or context for the document.
  • Author: Use your team or organization name if appropriate.

Keeping these properties up to date helps with version control, sharing, and accessibility.

Accessibility tips for document properties

  • Make sure the Title matches the file name and is meaningful.
  • Expand acronyms the first time you use them in the Title or Subject.
  • Check properties after exporting to PDF or other formats—they often carry over.
  • PDF Export Tip: In Word/PowerPoint, use File > Export > Create PDF/XPS > Options to ensure document properties are included when applicable.
  • Additional Properties: Consider using "Comments" and "Category" fields as supplemental properties for internal search if your organization uses them.

How properties improve discoverability

Search engines and internal file searches use document properties to help users find files. Well-chosen Titles and Keywords make your documents easier to locate and understand, especially for people using assistive technology.

Quick check before sharing

  • Review Title, Author, Subject.
  • Make sure the file name and Title are aligned.
  • Ask a colleague if the Title and Subject make sense without opening the file.
