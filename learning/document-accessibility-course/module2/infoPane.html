Why the Properties Pane matters

The Properties pane in Microsoft Word, PowerPoint, Excel, and Outlook lets you set key details about your file. These properties—like Title, Author, Subject, and Keywords—help users find, identify, and use your documents. They also support accessibility by providing screen readers and other assistive technologies with important context.

Where to find the Properties pane

  • Word, PowerPoint, Excel: Go to File > Info to view and edit properties.
  • Outlook: For attachments, right-click the file and choose Properties (some properties may be set in the original app).

The Properties pane is available in both desktop and web versions, though options may vary.

Key properties to fill in

  • Title: Use a clear, descriptive name for the document. Avoid codes or internal jargon.
  • Subject: Add a brief summary or context for the document.
  • Author: Use your team or organization name if appropriate.
  • Keywords: Add 3–5 relevant search terms to help others find your file.
  • Language: Set the main language of the document for correct spelling and pronunciation.
  • Comments: Add notes about the document’s purpose or status if needed.

Keeping these properties up to date helps with version control, sharing, and accessibility.

Accessibility tips for document properties

  • Make sure the Title matches the file name and is meaningful.
  • Expand acronyms the first time you use them in the Title or Subject.
  • Set the correct language, especially for documents with mixed content.
  • Remove “DRAFT” or other temporary notes from the Title before sharing.
  • Check properties after exporting to PDF or other formats—they often carry over.

How properties improve discoverability

Search engines and internal file searches use document properties to help users find files. Well-chosen Titles and Keywords make your documents easier to locate and understand, especially for people using assistive technology.

Quick check before sharing

  • Review Title, Author, Subject, Language, and Keywords.
  • Make sure the file name and Title are aligned.
  • Ask a colleague if the Title and Subject make sense without opening the file.
