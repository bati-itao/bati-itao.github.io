Module 5 – Outlook Email Accessibility Techniques

This module teaches Outlook-specific techniques for creating accessible emails. You'll learn why emails need unique accessibility steps and how to apply Outlook's built-in tools to create emails that work well with assistive technology.

By the end of this module, you'll be able to:

  • Set HTML as the default email format for accessibility
  • Structure bilingual emails with proper navigation
  • Apply and modify Outlook's built-in heading styles
  • Use accessible email templates effectively
