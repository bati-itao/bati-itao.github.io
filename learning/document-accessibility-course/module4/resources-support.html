ITAO Word templates

The IT Accessibility Office provides accessible Word templates designed specifically for Government of Canada use. These templates have accessibility features built in and comply with federal branding requirements.

Available templates

ITAO Accessible Word Templates

Templates include:

  • Accessible memo template
  • Accessible report template
  • Accessible letter template
  • Accessible briefing note template
  • Accessible meeting minutes template

Access templates: Visit the ITAO website at ESDC IT Accessibility Office and navigate to Resources → Document Templates

Template features

ITAO Word templates include:

  • Pre-configured heading styles: Proper hierarchy and accessible formatting
  • Accessible color schemes: Meets contrast requirements
  • Federal branding: FIP-compliant logos and formatting
  • Bilingual support: Templates available in English, French, and bilingual versions
  • Guidance notes: Instructions for maintaining accessibility
  • Tested for compliance: Verified against accessibility standards
  • Table styles: Pre-formatted accessible table designs

How to use ITAO templates

  1. Download the appropriate template from the ITAO website
  2. Save to your computer or organization's template library
  3. Create new document: File → New → select the ITAO template
  4. Replace placeholder text with your content
  5. Use the provided styles - don't create new ones or override formatting
  6. Follow any guidance notes included in the template
  7. Run Accessibility Checker before finalizing

Benefits of using ITAO templates

  • Save time on document setup
  • Ensure compliance with accessibility standards
  • Meet federal branding requirements
  • Consistency across your organization
  • Professional appearance
  • Easier to convert to accessible PDFs

Quick reference guides

Use these quick reference guides as reminders while working on Word documents.

Word Accessibility Quick Checklist

Document Structure

  • ☐ Use built-in heading styles (not manual formatting)
  • ☐ Only one Heading 1 per document
  • ☐ Logical heading hierarchy (no skipped levels)
  • ☐ Check structure using Navigation Pane
  • ☐ Avoid essential content in headers/footers

Tables

  • ☐ Header Row option enabled
  • ☐ No merged or split cells
  • ☐ No empty cells (or clearly labeled)
  • ☐ Use built-in table styles
  • ☐ Banded rows for tables with 5+ rows

Images and Non-Text Content

  • ☐ All images have alt text (or marked decorative)
  • ☐ Alt text is descriptive and concise (under 125 characters)
  • ☐ Long descriptions provided for complex visuals
  • ☐ FIP images have bilingual alt text where required

Text and Formatting

  • ☐ Sufficient color contrast (4.5:1 minimum)
  • ☐ Color not used as only way to convey information
  • ☐ Readable fonts and appropriate sizes
  • ☐ Meaningful link text (not "click here")

Additional Elements

  • ☐ Abbreviations and acronyms defined on first use
  • ☐ Multimedia has captions and transcripts
  • ☐ Document title property set
  • ☐ Document language set correctly

Final Check

  • ☐ Run Accessibility Checker
  • ☐ All errors fixed
  • ☐ Warnings addressed
  • ☐ Document tested with Navigation Pane

Common Keyboard Shortcuts

Action Shortcut
Apply Heading 1 Ctrl+Alt+1
Apply Heading 2 Ctrl+Alt+2
Apply Heading 3 Ctrl+Alt+3
Open Navigation Pane Ctrl+F or Alt, W, K
Open Styles pane Ctrl+Alt+Shift+S
Insert hyperlink Ctrl+K
Check accessibility Alt+R, A (then navigate)

Alt Text Quick Guide

Image Type What to Include
Simple image Brief description of what the image shows
Image with text All text visible in the image
Logo Organization name (bilingual if FIP)
Chart/graph Type of chart + what it shows + provide long description
Decorative image Mark as decorative (empty alt text)
Complex diagram Brief summary + provide detailed long description
