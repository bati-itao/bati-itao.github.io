Overview

Images can enhance documents by illustrating concepts, providing visual context, and breaking up text. However, images must be made accessible to users who cannot see them, including people who are blind, have low vision, or use screen readers. Alternative text (alt text) is the primary method for making images accessible in Word documents.

This section covers how to add meaningful alternative text and format images for maximum accessibility.

Why alternative text matters

Alternative text serves multiple important purposes:

For users with disabilities

  • Screen reader users: Alt text is read aloud to describe visual content
  • Users with low vision: Alt text provides context when images don't load or are unclear
  • Cognitive disabilities: Alt text can reinforce or clarify visual information

For all users

  • Describes images when they fail to load
  • Provides searchable text content
  • Improves document comprehension
  • Supports different reading preferences

Information begins

Legal requirement

Providing alternative text for images is often a legal requirement under accessibility legislation in many jurisdictions, including the ADA in the United States and AODA in Ontario, Canada.

Information ends

Adding alternative text in Word

Word provides several ways to add alternative text to images:

Method 1: Right-click context menu

  1. Right-click on the image
  2. Select Edit Alt Text
  3. Type your alternative text in the dialog box
  4. Click outside the dialog to save

Method 2: Format Picture pane

  1. Select the image
  2. Right-click and choose Format Picture
  3. Click the Layout & Properties icon
  4. Expand the Alt Text section
  5. Enter your alternative text

Method 3: Accessibility Checker

  1. Go to Review > Check Accessibility
  2. Click on images that need alt text
  3. Add alternative text in the provided field

Good example begins

Word's accessibility features

Recent versions of Word include an AI-powered feature that can suggest alternative text automatically. While these suggestions can be helpful, always review and edit them to ensure accuracy and relevance to your content.

Good example ends

Writing effective alternative text

Good alternative text is concise, accurate, and contextually relevant:

General guidelines

  • Keep alt text under 125 characters when possible
  • Be accurate and descriptive
  • Don't start with "Image of" or "Picture of"
  • Include relevant context and purpose
  • Avoid redundancy with surrounding text

Types of images and alt text approaches

Photographs
Describe the main subject and relevant details: "A woman presenting to a boardroom of six business professionals"
Icons and symbols
Describe the function or meaning: "Warning icon" or "Download button"
Diagrams
Describe the overall concept and key relationships: "Flowchart showing the approval process from submission to final decision"
Screenshots
Describe what's shown and its purpose: "Word ribbon showing the Insert tab with Table button highlighted"

Good example begins

Examples of effective alt text

  • Chart: "Bar chart showing 25% increase in sales from January to March 2023"
  • Photo: "Team of five developers collaborating around a computer screen"
  • Logo: "Company ABC logo"
  • Diagram: "Network diagram showing server connected to three workstations"

Good example ends

Decorative images

Some images are purely decorative and don't convey important information:

Identifying decorative images

Images are likely decorative if:

  • They're used for visual appeal only
  • They don't add information to the content
  • Removing them wouldn't affect understanding
  • They're background patterns or borders

Handling decorative images

For decorative images in Word:

  1. Right-click on the image
  2. Select Edit Alt Text
  3. Check the Mark as decorative option
  4. This tells screen readers to skip the image

Warning begins

Be careful with decorative marking

Only mark images as decorative if they truly add no informational value. When in doubt, provide alt text rather than marking as decorative.

Warning ends

Complex images and charts

Complex images may need additional description beyond alt text:

Approaches for complex images

  • Alt text: Brief overview of the image content
  • Long description: Detailed description in the document text
  • Data tables: Present chart data in accessible table format
  • Captions: Add numbered captions for reference

Charts and graphs

For charts and graphs:

  1. Provide alt text summarizing the main message
  2. Include the chart type and key data points
  3. Consider providing the raw data in table format
  4. Explain trends and significant findings in text

Good example begins

Example: Complex chart description

Alt text: "Line chart showing website traffic increasing from 10,000 to 25,000 visitors per month over six months"

Following text: "The chart shows steady growth in website traffic, with the most significant increase occurring between March and April, rising from 15,000 to 20,000 monthly visitors. This 33% jump coincides with our new marketing campaign launch."

Good example ends

Image placement and formatting

How you place and format images affects accessibility:

Image positioning

  • Use In Line with Text for most images
  • Avoid Behind Text or complex wrapping
  • Place images near relevant text
  • Ensure logical reading order is maintained

Captions and labels

  1. Right-click on your image
  2. Select Insert Caption
  3. Choose appropriate label (Figure, Table, etc.)
  4. Write descriptive caption text

Image quality and contrast

  • Use high-quality, clear images
  • Ensure sufficient contrast in image content
  • Avoid images with text unless necessary
  • If images contain text, include that text in alt text

Best practices

  • Add alt text to all meaningful images
  • Mark purely decorative images as decorative
  • Keep alt text concise but descriptive
  • Include relevant context and purpose
  • Don't repeat information from surrounding text
  • Use captions for complex images that need detailed description
  • Test with screen readers when possible
  • Provide data tables for charts and graphs
  • Use clear, high-quality images
  • Avoid using images for text content
  • Place images near related text content
  • Use consistent image formatting throughout the document
