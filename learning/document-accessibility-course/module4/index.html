Module 4 – Word Accessibility Techniques

This module teaches you to create accessible Microsoft Word documents using Word 365's built-in accessibility features. You'll learn Word-specific techniques for structure, navigation, tables, images, and accessibility tools that complement the general M365 accessibility principles covered in Module 2.

By the end of this module, you'll be able to:

  • Create properly structured Word documents with accessible headings and navigation
  • Build accessible tables using Word-specific features
  • Manage non-text content with appropriate descriptions and bilingual requirements
  • Use Word's accessibility checker and templates effectively

Lessons in this module

  1. Introduction
  2. Word-Specific Structure and Navigation
  3. Tables in Word
  4. Managing Non-Text Content
  5. Word-Specific Tools and Templates
  6. Additional Best Practices
  7. Resources and Support
