Module 4 – Word Accessibility Techniques

This module covers best practices for creating accessible Word documents, including the use of headings, styles, tables, images, hyperlinks, and accessibility tools to improve navigation and readability. You will learn how to structure documents logically, create accessible tables and forms, write meaningful hyperlinks, and use Word's built-in accessibility checker. These techniques ensure that your documents are fully accessible to users with disabilities while maintaining professional appearance and functionality.

  2. Using Word styles and formatting
  3. Writing accessible content
  4. Tables in Word documents
  5. Images and alternative text
  6. Hyperlinks and cross-references
  7. Accessibility checker and review tools
  8. Document templates and best practices
  9. Best practice
