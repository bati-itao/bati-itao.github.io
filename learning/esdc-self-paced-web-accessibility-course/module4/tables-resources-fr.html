Ressources des tableaux dans la boîte à outils de l'expérience Web (WET)

Le Guide de style de la BOEW pour les tableaux décrit les différents effets visuels et interactifs des noms de classe des feuilles de style CSS de la BOEW.

Le Plugiciel tables de la BOEW offre des fonctions de recherche, de tri, de filtrage, de pagination et d’autres fonctions avancées pour les tableaux.

Le validateur de tableau de la BOEW détecte les erreurs dans le balisage d’un tableau et génère un balisage id/headers. Le balisage généré ne conserve pas les attributs existants du tableau, qui doivent être restaurés manuellement dans le tableau valide. L’extension Firefox Complex Data Table Markup Toolbar est une solution de rechange au validateur de tableaux de la BOEW.
