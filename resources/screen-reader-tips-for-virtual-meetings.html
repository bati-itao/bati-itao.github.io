JAWS Tips for Microsoft Teams

For a list of MS Teams specific keyboard shortcuts and how-to’s see (link to MS Teams features page on github).

Action Description Key Combination
Mute and unmute MS Teams chat messages. JAWS Key + SPACEBAR then S
Currently available MS Teams keystrokes. INSERT + W

JAWS Useful Shortcuts

Starting a one to one chat

To start a one to one chat in MS Teams, type CTRL + N. Type a partial name in the search box and use Down Arrow to select a person. Press enter. Press enter a second time to open the chat.

Starting a one to many chat

To start a one to many chat in MS Teams, type CTRL + N. Type a partial name in the search box and use Down Arrow to select a person. Press enter. Continue adding new members until done. Press enter an extra time to open the chat.

MS Teams Command List

MS Teams uses the Search / Command box to allow users the ability to easily perform tasks using commands instead of mouse and curser actions. Provided below is a list of MS Teams commands. Users should keep in mind if a certain command isn't working for you, it could be because the organization has disabled a feature central to the command.

Pressing CTRL + E in MS Teams will put the focus in the command box. From the command box entering only forward slash ( / ) will display a list of available recognized commands. Using the up and down arrow keys allow you to move through the commands list to select the needed command.

Action Command
Set your status to away. /away
/busy Set your status to busy.
/call Call a phone number or Teams contact.
/dnd Set your status to do not disturb.
/files See your recent files.
/goto Go right to a team or channel.
/help Get help with Teams.
/join Join a team.
/keys See keyboard shortcuts.
/mentions See all your @mentions.
/org See someone's org chart.
/saved See your saved messages.
/testcall Check your call quality.
/unread See all your unread activity.
/whatsnew See what's new in Teams.
/whatsnew See what's new in Teams.
/who Ask Who a question about someone.
/wiki Add a quick note.

JAWS and NVDA Tips for Zoom meetings

JAWS specific keyboard shortcuts for Zoom

For a list of Zoom specific keyboard shortcuts and how-to’s see (link to Zoom features page on github).

Action Description Key Combination
Find out who is currently talking in a Zoom meeting. CTRL + SHIFT + T
View a list of JAWS keystrokes. INSERT + H
View a list of Zoom hot keys. INSERT + W
Enables or disables event alerts. ALT + WINDOWS
Repeats the last alert. ALT + WINDOWS + A
Reports if alerts are enabled or disabled. INSERT + TAB
Reads out the most recent chat messages. CTRL + 1 through CTRL + 0
